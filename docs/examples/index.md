# Examples

Runnable, standalone scripts under [`examples/`](https://github.com/AElOuassouli/my-rl-lib/tree/main/examples)
in the repository. Each page below embeds the actual script source, so what
you read here is exactly what you can run with `uv run python examples/<script>.py`.

- [SARSA with MLflow](sarsa-mlflow.md) — train SARSA on Windy Grid World with MLflow visualization.
- [Q-Learning Windy Grid World with MLflow](q-learning-windy-gridworld.md) — off-policy Q-Learning with full MLflow artifact logging (heatmaps, animations, registered model).
- [Metrics Tracking](metrics-tracking.md) — n-step SARSA with a metrics collector.
